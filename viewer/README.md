# Memori Viewer

The official interactive visualization UI for the [`memori`](https://crates.io/crates/memori) Rust benchmarking suite.  
Built with React, TypeScript, and Vite.

## Overview

Memori Viewer allows you to easily analyze and compare the benchmark results generated by the `memori` Rust crate.  
Simply load the output JSON file, and the viewer will automatically render beautiful, interactive charts of CPU cycles, instructions, execution time, and memory usage.

<details>
<summary>Japanese</summary>

## 概要

Memori Viewerを使用すると、Rust側の `memori` クレートが生成したベンチマーク結果を簡単に分析・比較できます。  
出力されたJSONファイルを読み込むだけで、CPUサイクル、命令数、実行時間、メモリ使用量などを視覚化したインタラクティブなチャートが自動的にレンダリングされます。

## Project Structure

To keep the development simple and maintainable, the source code is organized cleanly into standard React patterns:

- `src/components/` : Reusable UI components (Charts, Filter Controls, Layouts).
- `src/hooks/` : Custom React hooks for managing chart states and filtering logic.
- `src/utils/` : Pure functions for data parsing, JSON loading, and chart data formatting.
- `src/types/` : Strict TypeScript definitions mapped to the Rust benchmark outputs.

## プロジェクト構成

開発をシンプルかつ保守しやすくするため、標準的なReactのパターンに従って整理されています。

- `src/components/` : 再利用可能なUIコンポーネント（チャート、フィルター制御、レイアウトなど）。
- `src/hooks/` : グラフの状態管理やフィルタリングのロジックをカプセル化したカスタムフック。
- `src/utils/` : データ解析、JSONの読み込み、チャート用データの成形を行う純粋関数群。
- `src/types/` : Rust側の出力データ構造と厳密にマッピングされたTypeScriptの型定義。
